# IterateOverFiles
Iterate over folders and contained files to update them with a new column (CAGR calculation)
File structure:

      root
      |
      |__Folder1
      |  |-File1
      |  |-File2
      |  |_....
      |
      |__Folder2
      |  |-File1
      |  |-File2
      |  |_...
      |
      |__Folder2
      |  |-File1
      |  |-File2
      |  |_...
      |__...
